WhatsApp Web Contact Blur

WhatsApp Web Contact Blur helps protect your privacy when using WhatsApp Web in public or shared spaces.

The extension blurs your contacts list by default, keeping names and profile pictures hidden.
Contacts become visible on mouse hover or click, so you can quickly check messages without exposing your entire chat list.
